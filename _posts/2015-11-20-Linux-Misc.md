###yum transcation hangs

su -c 'package-cleanup --problems'
su -c 'yum --setopt=tsflags=noscripts remove zarafa*'

`su -c 'rpm -e --noscripts zarafa*'`

####Linux 使用key验证后无密码登录维护怎么处理Give root password for maintenance

* select boot item
* 'e' kernel line
* add 'init=/bin/bash'
* 'b' to boot kernel
* mount -o remont,rw /
* passwd root
* reboot
